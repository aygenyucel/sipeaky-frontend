# Sipeaky

A real-time study platform where users can create and join interactive rooms for language practice and focused sessions.

Create a room, share the link, and start studying together instantly.

---

## 🚀 Live Demo

https://sipeaky.vercel.app

---

## ✨ Features

* Create study rooms with:

  * Language selection
  * Level (Beginner, Intermediate, Advanced, Native)
  * Room capacity
* Join rooms and communicate via:

  * Text chat
  * Video communication (WebRTC)
* Real-time user updates using Socket.io
* Shareable room links to invite others instantly
* Simple and distraction-free study environment

---

## 🛠 Tech Stack

* Frontend: React, Redux
* Backend: Node.js, Express
* Real-time: Socket.io, WebRTC (PeerJS)
* Database: MongoDB

---

## ⚙️ Setup

```bash
npm install
npm run dev
```

## 💡 About the Project

This project was built to explore real-time communication and WebRTC-based interactions in a social learning environment.

Users can create rooms by selecting language, level, and capacity, then invite others via a unique link to join and interact in real time.
